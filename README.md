# Secure Employee Management REST API

A backend REST API built using Java and Spring Boot for managing employees with secure role-based access control.

## Features
- Full CRUD operations
- Spring Security authentication & authorization
- Role-based access control (EMPLOYEE, MANAGER, ADMIN)
- MySQL database integration
- PATCH support for partial updates
- REST API testing with Postman

## Technologies Used
- Java
- Spring Boot
- Spring Security
- Spring Data JPA
- MySQL
- Maven
- Postman

## API Endpoints
- GET /employees
- GET /employees/{id}
- POST /employees
- PUT /employees/{id}
- PATCH /employees/{id}
- DELETE /employees/{id}
